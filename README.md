# Landing Page - Nutricionista

Landing page profissional para nutricionista desenvolvida em Flutter com integração ao WhatsApp.

## Recursos

- Design responsivo (Desktop e Mobile)
- Apresentação profissional
- Serviços oferecidos
- Planos de consulta e acompanhamento
- Depoimentos de pacientes
- Integração com WhatsApp
- Interface moderna e acolhedora

## Como executar

```bash
flutter pub get
flutter run -d chrome
```
